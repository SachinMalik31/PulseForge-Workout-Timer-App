# 💪 PulseForge – Workout Timer App

## 🚀 Overview
PulseForge is a modern workout timer mobile application built with React Native and Expo, designed to transform workouts from a manual process into a system-controlled experience.

## ⚙️ Key Idea
Instead of requiring users to track sets, rest intervals and transitions manually, PulseForge uses a **state-driven execution engine** to automate the entire workout lifecycle, from start to finish.

## 🔔 Features
- ⏱️ Automated workout timer with real-time execution  
- 🔄 Seamless exercise & rest transitions  
- 🔊 Audio cues & voice guidance (hands-free workouts)  
- 📳 Haptic feedback for transitions  
- 📊 Workout history & progress tracking  
- ☁️ Firebase integration for data persistence  
- 🤖 AI-powered coaching for personalized guidance  

## 🎯 Goal
To reduce cognitive load and allow users to fully focus on their workout while the system handles timing, tracking and flow.
