# 💪 PulseForge – Workout Timer App

## 🚀 Overview
PulseForge is a modern workout timer mobile application built with React Native and Expo, designed to transform workouts from a manual process into a system-controlled experience.

## ⚙️ Key Idea
Instead of requiring users to track sets, rest intervals and transitions manually, PulseForge uses a **state-driven execution engine** to automate the entire workout lifecycle, from start to finish.

## 🔔 Features
- ⏱️ Automated workout timer with real-time execution  
- 🔄 Seamless exercise & rest transitions  
- 🔊 Audio cues & voice guidance (hands-free workouts)  
- 📳 Haptic feedback for transitions  
- 📊 Workout history & progress tracking  
- ☁️ Firebase integration for data persistence  
- 🤖 AI-powered coaching for personalized guidance  

## 🎯 Goal
To reduce cognitive load and allow users to fully focus on their workout while the system handles timing, tracking and flow.

### This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.
